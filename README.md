# Airbnb Price & Revenue Analysis Dashboard

## Overview
This project presents an interactive Tableau dashboard analyzing Airbnb pricing and revenue trends across different locations and bedroom types.

## Tools Used
- Tableau
- Data Visualization
- Data Analysis

## Key Features
- Average price per bedroom analysis
- Price distribution across zipcodes
- Revenue trends over time
- Interactive dashboard with multiple visualizations

## Insights
- Identified pricing differences based on bedroom count
- Analyzed regional price variations
- Observed revenue trends over time
